# 科技系统设计

## 1. 设计目标
- 以矩阵驱动技术解锁，形成“科研推进 -> 能力开放 -> 产线升级”闭环。
- 技术效果可组合叠加，作用于建筑、物流、能源、战斗、戴森球。

## 2. 科技模型
- 科技定义字段：
  - `id`、名称、分类、层级
  - 研究成本（矩阵配方与研究点）
  - 前置科技
  - 解锁内容（建筑/配方/能力）
- 玩家科技状态：
  - 已解锁集合
  - 当前研究目标
  - 研究进度与历史记录

## 3. 矩阵体系
- 电磁矩阵（蓝）
- 能量矩阵（红）
- 结构矩阵（黄）
- 信息矩阵（紫）
- 引力矩阵（绿）
- 宇宙矩阵（白）

## 4. 研究流程
- 玩家下达开始研究命令。
- 校验前置科技与研究成本。
- Tick 内按研究速率推进进度，完成后解锁效果并发布事件。
- 可接入执行体研究加成与建筑研究增益。

## 5. 关键解锁方向
- 物流升级：带速、分拣、站点容量、星际运输。
- 生产升级：制造速度、增产、专精配方。
- 能源升级：高效发电、储能、接收站效率。
- 戴森升级：应力上限、结构效率、光子链路。
- 战斗升级：武器伤害、射程、防御与侦测能力。
