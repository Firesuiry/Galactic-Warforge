# 能源与电网系统设计

## 1. 设计目标
- 建立统一电网拓扑，聚合发电、输电、储能、耗电行为。
- 在 Tick 内完成供需平衡与优先级分配。
- 为高阶系统（戴森球接收、护盾、防御炮塔）提供稳定能源基础。

## 2. 电网组成
- 发电端：风电、火电、太阳能、地热、聚变、人造恒星。
- 输配端：电塔、无线输电、卫星配电站。
- 储能端：蓄电器与能量枢纽。
- 负载端：生产建筑、物流站、防御建筑、研究建筑。

## 3. 拓扑模型
- 节点：建筑电力节点或连接器节点。
- 边：可导通电力连接关系，支持动态重建。
- 连通域：形成独立电网，分别进行功率聚合与分配。

## 4. 结算规则
- 供需聚合：每个连通域统计总发电与总负载。
- 优先级分配：保障关键负载（基础物流/防御）优先供电。
- 储放电策略：
  - 富余时充电
  - 紧缺时放电
- 极端短缺：按优先级停机并发出缺电事件。

## 5. 与戴森系统联动
- 射线接收站可将轨道能量转化为电力或临界光子。
- 戴森结构产能提高后，优先用于电网稳定与终局制造链供能。
- 护盾与高阶武器属于高波动负载，需预留储能缓冲。
