# 戴森球计划：三层分级（行星内 / 恒星系 / 星系级）

说明：依据 `docs/戴森球计划科技树.md` 与 `docs/dyson-sphere-program-buildings.md` 进行分层，便于后续分级实现。分层按“功能覆盖范围 + 关键依赖”归类，少量科技为方便实现被归入其主要应用层级。

分层规则（简要）：
- 行星内：不依赖行星间物流、轨道结构或翘曲的单行星闭环能力。
- 恒星系：涉及行星间物流、轨道结构、戴森蜂群/戴森球、气态行星采集、星系内空间作战。
- 星系级：跨恒星系行动（翘曲/引力）与反物质/宇宙矩阵相关终局能力。
