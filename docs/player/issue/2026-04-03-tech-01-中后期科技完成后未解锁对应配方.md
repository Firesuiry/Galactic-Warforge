# 多项中后期科技完成后未解锁对应配方

- 状态：已修复 / 已过时
- 首次记录：2026-04-03
- 来源：`docs/player/已知问题与回归.md` -> `2026-04-03 当时问题列表`
- 类别：科技树 / 配方解锁 / 历史问题

## 问题描述

多项戴森中后期科技虽然研究完成，但并没有解锁对应配方，导致真实物料链断开。

## 当时现象

- `high_strength_crystal.unlocks = null`
- `titanium_alloy.unlocks = null`
- `quantum_chip.unlocks = null`
- `lightweight_structure` 只剩 `dyson_component`，没有 `frame_material`
- `/catalog` 中缺失：
  - `titanium_crystal`
  - `titanium_alloy`
  - `frame_material`
  - `quantum_chip`
  - `small_carrier_rocket`

## 影响

- 科技树可以点到后期，但不少真实 DSP 物料链无法搭建
- 戴森材料、量子芯片、火箭链路全部断开

## 临时绕过

- 当时只能继续用不消耗材料的 `build_dyson_*` 入口验证结构结算

## 当前状态

- 后续文档已将这类“科技树不可达建筑与空科技节点”标记为 T103 收口问题
