# 默认新局无法仅靠 Web 完成最小科研起步链

- 状态：待处理
- 首次记录：2026-04-10
- 来源：`docs/player/已知问题与回归.md` -> `2026-04-10 Web 试玩`
- 类别：Web / 玩法主线 / 研究

## 问题描述

默认新局虽然能通过 `client-web` 造出第一台 `wind_turbine` 和第一台空 `matrix_lab`，但无法继续只靠 Web 走通 `electromagnetism` 的真实研究闭环。

## 复现步骤

1. 用 `p1 / key_player_1` 登录 `client-web`
2. 打开 `planet-1-1`
3. 在“命令”页依次执行：
   - `build 3 2 wind_turbine`
   - `build 2 3 matrix_lab`
4. 尝试继续通过 Web 推进 `electromagnetism`

## 实际现象

- 当前 Web 命令面板只提供：`扫描 / 建造 / 移动 / 物流站配置 / 物流槽位配置 / 研究 / 拆除`
- 没有 `transfer` 入口，玩家无法把开局背包里的 `electromagnetic_matrix = 50` 装进研究站
- Web 内点击“开始研究”后，前端先显示 `accepted, will execute at next tick`
- authoritative 结果随后失败为：`VALIDATION_FAILED: missing electromagnetic_matrix in research labs`

## 影响

- 默认新局会在开局几步内卡死在第一门科研前
- “主要用 Web 进行试玩”当前并不成立
- Web 与 `docs/player/玩法指南.md` 的最小起步链不一致

## 临时绕过

- 改用 `client-cli transfer`
- 或者直接调用服务端 API 完成 `transfer`
- 纯 Web 当前无解
