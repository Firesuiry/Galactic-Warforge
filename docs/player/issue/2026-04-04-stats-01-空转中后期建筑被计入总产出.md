# `/state/stats` 会把空转中后期建筑计入总产出

- 状态：待复核 / 疑似仍需关注
- 首次记录：2026-04-04
- 来源：`docs/player/已知问题与回归.md` -> `2026-04-04 晚间追加复测`
- 类别：统计口径 / 产线监控 / Web 概览

## 问题描述

`/state/stats` 的生产统计曾把没有实际产出的中后期建筑计入 `production_stats.total_output`。

## 复现样例

- `recomposing_assembler (b-51)` 没有 `recipe_id`
- `self_evolution_lab (b-52)` 没有 `recipe_id`
- `vertical_launching_silo (b-35)` 长期 `input_shortage = true`

## 实际现象

`GET /state/stats` 仍返回：

- `production_stats.total_output = 6`
- `by_building_type.recomposing_assembler = 2`
- `by_building_type.self_evolution_lab = 3`
- `by_building_type.vertical_launching_silo = 1`

## 影响

- 玩家、Web 总览和开发者都会误判真实产能
- 中后期排障与经济判断会被污染
