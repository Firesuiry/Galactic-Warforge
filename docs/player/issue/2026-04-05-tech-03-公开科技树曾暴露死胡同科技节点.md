# 公开科技树曾暴露死胡同科技节点

- 状态：已修复（见 T103 收口）
- 首次记录：2026-04-05
- 来源：`docs/player/已知问题与回归.md` -> `2026-04-05 戴森深度试玩补测 → 本轮问题收口结果`
- 类别：科技树 / 目录口径

## 问题描述

公开科技树曾暴露一批当前版本并不打算继续提供给玩家推进的死胡同科技节点，导致科技树可见范围与真实可玩范围不一致。

## 收口后口径

以下节点已不再继续暴露给玩家：

- `casimir_crystal`
- `crystal_explosive`
- `crystal_shell`
- `proliferator_mk2`
- `proliferator_mk3`
- `reformed_refinement`
- `super_magnetic`
- `supersonic_missile`
- `titanium_ammo`
- `wave_interference`
- `xray_cracking`

同时保留桥接科技并通过 `leads_to` 表达后继价值。

## 影响

- 修复前玩家会在公开科技树里看到无法形成正向收益的节点，认知成本很高
