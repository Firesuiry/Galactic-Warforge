# 移动端行星页几乎不可玩

- 状态：待处理
- 首次记录：2026-04-10
- 来源：`docs/player/已知问题与回归.md` -> `2026-04-10 Web 试玩`
- 类别：Web / 移动端适配 / UI

## 问题描述

`planet` 页在约 `390x844` 窄屏下虽然能打开，但主地图和三栏布局会严重变形，核心玩法区域几乎失去可用性。

## 复现步骤

1. 在移动端尺寸约 `390x844` 下打开 `planet-1-1`
2. 观察地图、摘要面板与右侧操作区布局

## 实际现象

- 主地图区域被压成狭长条带
- 左中右三栏只是简单纵向堆叠
- 页面退化成一长串摘要卡片和表单
- 地图观察、定位和命令联动难以使用

## 影响

- 行星页在移动端基本失去“战略地图”意义
- 并非只是操作不方便，而是核心信息载体失效

## 临时绕过

- 暂时仅建议使用桌面端
