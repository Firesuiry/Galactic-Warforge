# 戴森球计划：建筑与功能一览（中文名 + 分类）

说明：建筑中文名称与分类参照 DSP WIKI（BWIKI）。功能为简要总结，保持与上一版一致。

整理日期：2026-03-17

## 电力
| 建筑（中文 / 英文） | 功能（简要） |
| --- | --- |
| 电力感应塔（Tesla Tower） | 电力传输设施，扩展电网覆盖范围。 |
| 无线输电塔（Wireless Power Tower） | 远距离无线供电，可从电网取电并为机甲充电。 |
| 卫星配电站（Satellite Substation） | 大范围配电设施，提供更远距离的电力覆盖。 |
| 风力涡轮机（Wind Turbine） | 将风能转为电能，发电量受星球风能利用率影响。 |
| 火力发电厂（Thermal Power Plant） | 消耗可燃燃料发电（如煤），效率与燃料消耗相关。 |
| 太阳能板（Solar Panel） | 利用光照发电，夜间停止；可配合蓄电器储能。 |
| 地热发电站（Geothermal Power Station） | 利用岩浆发电，输出与岩浆浓度相关。 |
| 微型聚变发电站（Mini Fusion Power Plant） | 消耗氘燃料棒进行受控聚变发电。 |
| 能量枢纽（Energy Exchanger） | 能量交换器：将电网富余电充入空蓄电器，或从满蓄电器放电。 |
| 蓄电器（Accumulator） | 电力储能设施：电力富余时充电，缺电时放电。 |
| 蓄电器（满）（Accumulator (Full)） | 满电蓄电器，可在缺电时放电或放入能量交换器放电。 |
| 射线接收站（Ray Receiver） | 接收戴森蜂群/戴森球能量并入电网，或产出临界光子。 |
| 人造恒星（Artificial Star） | 人工太阳设施（描述为正负质子束在约束球内形成）。 |

## 采集
| 建筑（中文 / 英文） | 功能（简要） |
| --- | --- |
| 采矿机（Mining Machine） | 覆盖矿脉采矿，覆盖越多矿脉采速越快。 |
| 大型采矿机（Advanced Mining Machine） | 高效激光采矿，覆盖范围大，矿石可由传送带或物流无人机输出。 |
| 抽水站（Water Pump） | 从海洋抽取液体（含硫酸海）。 |
| 原油萃取站（Oil Extractor） | 从油井抽取原油，速度受油井产量与资源利用率影响。 |
| 轨道采集器（Orbital Collector） | 从气态巨行星采集资源，运行需消耗采集的燃料。 |

## 物流
| 建筑（中文 / 英文） | 功能（简要） |
| --- | --- |
| 传送带（Conveyor Belt MK.I） | 基础传送带，用于运输货物。 |
| 高速传送带（Conveyor Belt MK.II） | 更快的传送带，吞吐更高。 |
| 极速传送带（Conveyor Belt MK.III） | 最高速度与吞吐的传送带。 |
| 四向分流器（Splitter） | 分流或平衡多条传送带，支持过滤与优先级。 |
| 自动集装机（Automatic Piler） | 自动堆叠器，将传送带上两份货物合并，提高带宽。 |
| 流速监测器（Traffic Monitor） | 监控传送带流量并按条件触发报警。 |
| 喷涂机（Spray Coater） | 为传送带货物喷涂增产剂/加速剂，提高后续产出或速度。 |
| 物流配送器（Logistics Distributor） | 安装在仓顶，物流机器人可向机甲或其他分发器配送物品。 |
| 行星内物流运输站（Planetary Logistics Station） | 行星内物流枢纽，设置供需后由物流无人机配送（限同星球）。 |
| 星际物流运输站（Interstellar Logistics Station） | 星际物流枢纽，实现跨星球运输。 |

| --- | --- |

| --- | --- |

| 位面熔炉（Plane Smelter） | 高效冶炼/提纯设备，速度更快。 |
| 负熵熔炉（Negentropy Smelter） | 负熵冶炼炉，将杂质矿物复原为元素态。 |

| 分馏塔（Fractionator） | 将氢分馏得到氘，氢从一侧进入，产出氢与氘。 |

| 量子化工厂（Quantum Chemical Plant） | 量子化工设备，强化反应过程，用于高级化工配方。 |
| 微型粒子对撞机（Miniature Particle Collider） | 粒子对撞产生新粒子，常用于稳定氘或反物质，耗能高。 |

## 防御
| 建筑（中文 / 英文） | 功能（简要） |
| --- | --- |
| 高斯机枪塔（Gauss Turret） | 电磁炮塔，对低空单位具高杀伤。 |
| 导弹防御塔（Missile Turret） | 导弹炮塔，可锁定地面与太空目标，进行多目标防御。 |
| 聚爆加农炮（Implosion Cannon） | 重型爆炸炮塔，造成范围爆炸伤害，但无法攻击近距离目标。 |
| 高频激光塔（Laser Turret） | 激光炮塔，持续高温伤害，耗能较高。 |
| 磁化电浆炮（Plasma Turret） | 等离子炮塔，发射高能等离子束，适合太空目标。 |
| 近程电浆塔（SR Plasma Turret） | 地面高温等离子炮塔，近距高伤但耗能大且脆弱。 |
| 战场分析基站（Battlefield Analysis Base） | 战场分析与指挥核心，可派遣战机、防御与自动修复，并收集敌军残骸情报/资源。 |
| 干扰塔（Jammer Tower） | 干扰塔，周期性减速并降低敌方射速，消耗电磁胶囊。 |
| 信号塔（Signal Tower） | 战术信号塔，诱导黑雾改变进攻方向并加速其集结；可与导弹炮塔共享信息扩展射程。 |
| 行星护盾发生器（Planetary Shield Generator） | 生成行星护盾抵御来自太空的攻击，耗能大且数量有限。 |
