# T062 管线与建筑 IO 缓冲对接

## 需求细节
- 将管线网络与建筑 IO 缓冲对接，支持输入/输出端点。
- 定义端点拉取/推送策略与容量约束。
- 确保与物流/建筑运行时节拍一致，避免重复扣减或超发。
- 增加必要的联调与集成测试用例。

## 实现记录
- 新增管线 IO 结算流程，基于流体端口与管线节点缓存进行输入/输出对接。
- 端口按流体物品白名单筛选，输出时遵循节点容量与同种流体约束，输入时避免超收并支持回滚。
- Tick 流程中在建筑 IO 后加入管线 IO，确保缓冲结算顺序一致。
- 增加管线 IO 的联调测试，覆盖管线到建筑与建筑到管线的双向传输。
