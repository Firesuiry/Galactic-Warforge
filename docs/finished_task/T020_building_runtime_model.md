# T020 建筑功能挂载与基础运行参数模型

## 需求细节
- 建筑功能模块化挂载：生产、采集、运输、仓储、能量、研究等功能可组合。
- 基础运行参数模型：功耗、产能、占格、连接点、输入输出口、工作状态。
- 运行参数的通用接口，支持后续物流、电网、生产链集成。
- 与建筑定义结构的关联与校验规则。

## 前提任务
- T019 建筑分类与规格定义

## 实现结果
- 新增建筑运行参数模型与模块化功能挂载（生产/采集/运输/仓储/能量/研究/战斗），包含功耗、产能、占格、连接点、输入输出口与工作状态。
- 构建运行参数目录并与建筑规格关联校验，默认运行参数从建筑规格占格派生，支持按建筑类型覆盖。
- 引入建筑运行档案 `BuildingProfile`，统一生成建筑 HP/视野与运行参数，并在建造/升级/结算逻辑中切换到运行参数模型。
- 更新客户端类型与展示字段，服务端 API 文档补充建筑 `runtime` 结构说明。
