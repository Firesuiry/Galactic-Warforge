# T031 生产加成与喷涂集成到建筑

## 需求细节
- 将喷涂/加成系统与生产建筑运行流程集成。
- 支持加速与增产规则的配置与结算，明确与功耗/产能的联动。
- 喷涂影响范围、消耗规则与优先级策略。
- 与现有建筑运行参数与功能挂载保持一致，避免适配层。

## 前提任务
- 无

## 实现结果
- 新增生产周期解析器 `ResolveProductionCycle`，将配方、建筑运行参数、喷涂状态统一结算，输出加成结果、能耗/发电总量与有效吞吐。
- 生产加成结算只收集配方输入物品的喷涂状态，按配置叠加规则选择等级，并在成功应用时扣减喷涂次数；不足则回退为无加成并保持输入不变。
- 明确喷涂优先级与消耗策略：仅消耗命中等级的喷涂来源，其余等级保持不变；支持多堆叠喷涂次数合计扣减。
