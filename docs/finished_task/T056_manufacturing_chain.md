# T056 制造链

## 需求细节
- 中间件制造：电路板、马达、微晶元件等常见制造件。
- 高阶制造：粒子容器、湮灭约束球等复杂组件。
- 配方包含多输入多输出，支持并行制造与产线吞吐统计。
- 与建筑生产模型对接，确保制造链可被生产建筑执行。

## 实现记录
- 新增马达与湮灭约束球物品定义，补齐制造链的中间件与高阶组件。
- 增加马达与湮灭约束球配方，均挂载到制造台生产模块，满足多输入制造要求。
- 制造链沿用现有生产周期与吞吐统计，保持多输出配方支持（燃料棒回收等配方仍提供多输出示例）。
