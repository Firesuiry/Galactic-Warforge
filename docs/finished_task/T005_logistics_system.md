# T005 物流系统

## 需求细节
- 传送带网络：占格、方向、吞吐、堆叠、堵塞与溢出处理。
- 分拣器规则：速度、距离、过滤、优先级。
- 仓储规则：容量、优先输入输出、互连与缓存。
- 行星内物流：无人机容量、航速、起降、站点供需匹配。
- 星际物流：运输船容量、航速、能耗、翘曲驱动与翘曲器消耗。
- 轨道采集：气态行星采集产出、轨道站挂载与运输。
- 物流调度：需求预测、超额供给、最短路或最低成本策略。
