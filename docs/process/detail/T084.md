# T084 侦测与防御系统 - 架构设计

## 1. 概述

侦测与防御系统提供敌人的探测、追踪和防御能力，包括信号塔、雷达、防御建筑等。

## 3. 信号塔

### 3.1 功能

- 诱导敌人改变进攻方向
- 扩展玩家视野范围
- 消耗电磁胶囊

```go
// SignalTowerEffect 信号塔效果
type SignalTowerEffect struct {
    RangeBonus      float64 `json:"range_bonus"`       // 视野范围加成
    RedirectChance float64 `json:"redirect_chance"`    // 重定向几率
    EnergyCost     int     `json:"energy_cost"`       // 能量消耗
}

// ApplySignalTowerEffect 应用信号塔效果
func ApplySignalTowerEffect(tower *DefenseBuildingState, enemies []EnemyForce) {
    for i := range enemies {
        if calculateDistance(tower.Position, enemies[i].Position) <= tower.Range {
            // 重定向几率
            if rand.Float64() < tower.RedirectChance {
                enemies[i].TargetPlayer = "" // 清除目标
            }
        }
    }
}
```

## 4. 雷达系统

### 4.1 功能

- 探测敌人位置
- 识别敌人类型和数量
- 追踪敌人移动

```go
// RadarScan 雷达扫描
func RadarScan(radar *DefenseBuildingState, allEnemies []EnemyForce) []EnemyIntel {
    results := make([]EnemyIntel, 0)

    for _, enemy := range allEnemies {
        if calculateDistance(radar.Position, enemy.Position) <= radar.Range {
            intel := EnemyIntel{
                EnemyID:    enemy.ID,
                Type:      string(enemy.Type),
                Position:  enemy.Position,
                Strength:  enemy.Strength,
                LastSeen:  currentTick,
            }
            results = append(results, intel)
        }
    }
    return results
}
```

## 5. 防御建筑

### 5.1 炮塔

```go
// TurretStats 炮塔属性
type TurretStats struct {
    Damage        int     `json:"damage"`         // 伤害
    FireRate      float64 `json:"fire_rate"`       // 射速 (ticks/发)
    Range        float64 `json:"range"`          // 射程
    AmmoConsume  int     `json:"ammo_consume"`   // 每发弹药消耗
}

// ProcessTurretAttack 处理炮塔攻击
func ProcessTurretAttack(turret *DefenseBuildingState, target *EnemyForce) (damage int, success bool) {
    if !turret.IsActive || turret.AmmoCount <= 0 {
        return 0, false
    }

    distance := calculateDistance(turret.Position, target.Position)
    if distance > turret.Range {
        return 0, false
    }

    // 计算伤害
    damage = turret.Damage
    if target.ShieldLevel > 0 {
        shieldDamage := float64(damage) * 0.3
        target.ShieldLevel -= shieldDamage
        damage = int(float64(damage) * 0.7)
    }

    target.HP -= damage
    turret.AmmoCount -= turret.AmmoConsume

    return damage, true
}
```
