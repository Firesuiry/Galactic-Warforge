# T055 冶炼链

## 需求细节
- 基础矿物冶炼：铁矿、铜矿、硅石、钛石、石材。
- 支持基础配方产出：铁锭、铜锭、硅块、钛锭、石材。
- 冶炼配方的耗时、耗能与产出数量可配置。
- 与现有物品/配方系统对接，冶炼产物可进入后续制造链。

## 实现记录
- 为冶炼配方增加能耗字段，并在生产周期中纳入能耗计算，保持耗时/耗能/产出可配置。
- 冶炼配方改为由熔炉体系执行（弧/平面/负熵熔炉），与现有物品与配方系统对接。
- 补充三类熔炉的运行时生产模块与能耗参数，设为可建造并配置基础建造成本。
