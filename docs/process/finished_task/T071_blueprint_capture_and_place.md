# T071 蓝图生成与放置

## 需求细节
- 选区复制：从选区生成蓝图，包含建筑参数与相对坐标。
- 放置变换：支持偏移与旋转，生成放置后的世界坐标。
- 对齐与边界：处理跨区块/网格对齐，确保放置范围可计算。
- 失败收集：对非法坐标或缺失数据给出可汇总的错误信息。

## 完成情况
- 新增蓝图选区捕获与放置变换逻辑，支持边界校验、偏移/旋转与问题汇总。
- 捕获逻辑包含建筑参数提取（传送带/分拣器/仓储/物流站配置）与部分占用检测。
- 放置逻辑输出世界坐标、旋转后边界与方向修正，提供可复用的放置结果结构。
