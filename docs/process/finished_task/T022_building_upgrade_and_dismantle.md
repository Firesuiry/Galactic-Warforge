# T022 建筑升级与拆除

## 需求细节
- 建筑升级链：前后等级映射、升级成本、升级时长与条件。
- 拆除与回收：回收率、返还物品、拆除规则与限制。
- 与建筑定义/运行状态的联动规则（停机、占格释放、资源返还时机）。

## 前提任务
- T019 建筑分类与规格定义

## 实现结果
- 新增建筑升级/拆除规则结构，定义允许性、最大等级、耗时、返还率与停机约束；提供默认规则并计算升级成本与拆除返还。
- 升级/拆除执行改为可选耗时作业，作业期间建筑暂停运行，完成时升级生效或拆除释放占格并返还资源；执行体并发上限会计入进行中的建筑作业。
- 建筑实例新增 `job` 字段用于回传作业状态；服务端 API 文档与客户端类型同步补充。
