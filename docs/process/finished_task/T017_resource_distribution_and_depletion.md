# T017 资源分布与枯竭规则

## 需求细节
- 资源分布：矿脉类型、含量、分布形态、稀有资源点。
- 资源再生与枯竭规则（矿脉耗尽、油井衰减等）。
- 资源点可序列化并与地图坐标关联。

## 前提任务

## 实现结果
- 新增资源点模型（类型、行为、产出、坐标、稀有标记、簇 ID），资源点与地图坐标序列化绑定到行星。
- 资源生成加入簇状分布与稀有资源概率，资源点仅落在可建格。
- 资源枯竭与再生规则落地：矿脉有限耗尽、油井产出衰减、可再生资源按 tick 回补。
- 采矿建筑需建在资源点上，采集时扣减资源点状态并反映到行星视图。
