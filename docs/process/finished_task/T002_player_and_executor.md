# T002 玩家与执行体（上帝模式）

## 需求细节
- 玩家身份、队伍关系、权限与命令鉴权。
- 玩家以“上帝模式”进行全局指令下达，不直接扮演机甲单位。
- 开局默认提供一个无额外能耗的全能机器人作为执行体。
- 全能机器人按玩家命令执行建造、采集、拆除、搬运等操作。
- 执行体能力参数：建造效率、操作范围、并发任务能力、研究辅助加成。

## 前提任务
- T001 世界与地图
- T014 服务端接口约束与命令驱动

## 完成情况

- 开局为每位玩家生成执行体单位（executor），记录执行体能力参数并绑定到玩家状态。
- `build/produce/upgrade/demolish` 增加执行体范围与并发限制校验。
- 攻击与炮塔自动攻击增加同队伍免伤判定。
