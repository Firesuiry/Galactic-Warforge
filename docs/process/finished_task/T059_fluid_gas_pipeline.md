# T059 液体与气体管线

## 前提任务
- T060 液体与气体基础模型
- T061 管线网络与分流合流规则
- T062 管线与建筑 IO 缓冲对接

## 需求细节
- 管线基础参数：流速、压力、容量。
- 分流与合流规则：优先级、均分、溢出策略。
- 支持多段管线网络的传输损耗或衰减模型（可选）。
- 与物流/建筑 IO 缓冲对接，形成端到端流体传输。

## 实现记录
- 管线基础参数与分流/合流规则沿用既有模型与流量计算实现。
- 新增管线流量结算 `settlePipelineFlow`，在 Tick 中于建筑 IO 后、管线 IO 前推进管线网络流量。
- 当任一管线段配置衰减系数时自动启用衰减结算，保证多段管线端到端传输可用。
