# T041 分拣器规则与调度

## 需求细节
- 分拣器速度与距离：基于等级或配置的取放节拍与最大距离。
- 过滤规则：按物品/标签过滤的匹配与拒绝策略。
- 优先级规则：多目标/多来源时的优先级与调度顺序。
- 与传送带交互：分拣器对传送带物品的取放时机与一致性约束。
