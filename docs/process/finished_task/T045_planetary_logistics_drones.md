# T045 行星物流无人机

## 需求细节
- 无人机实体：容量、航速、起降、站点绑定与占用。
- 简化航线/移动时间计算（不含全局调度）。

## 交付物
- 服务端无人机运行状态与基础移动逻辑实现。
- 单元测试覆盖容量、起降流程、航速影响到达时间。
- 若涉及服务端 API 行为变更，更新 `docs/dev/服务端API.md`。

## 实现结果
- 新增行星物流无人机状态模型（容量、航速、货舱、飞行阶段、航线/剩余 Tick），提供起飞与载货/卸货接口及航线时间计算。
- 世界状态加入无人机注册表与站点容量校验，物流站点补充无人机容量配置。
- 新增无人机结算逻辑，按起飞/飞行/降落阶段推进并更新位置。
- 新增测试覆盖载货容量、航速影响到达时间、起降流程与站点容量占用。
