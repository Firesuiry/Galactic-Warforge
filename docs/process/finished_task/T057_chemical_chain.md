# T057 化工链

## 需求细节
- 化工产物：塑料、石墨、硫酸、石墨烯、碳纳米管。
- 支持液体/气体原料与固体原料混合配方。
- 化工配方的耗时、耗能与产出数量可配置。
- 化工产物可进入制造链与高级链使用。

## 实现记录
- 新增硫酸与碳纳米管物品定义，补齐化工链材料数据并标记液体/固体形态。
- 增加硫酸、石墨烯（石墨+硫酸）与碳纳米管配方，化工配方统一挂载到化工厂/量子化工厂并补充能耗配置。
- 为化工厂/量子化工厂补齐可建造成本与运行时生产模块，支持化工配方生产。
