# T047 发电类型与发电模型

## 需求细节
- 发电建筑分类：风能、太阳能、火力、聚变、人工恒星，统一发电接口。
- 发电量计算：基础功率、环境系数、燃料消耗（如适用）。
- 发电输出：将发电量写入电力流/电网输入数据结构。
- 运行周期：按 tick/step 更新发电与燃料消耗。

## 交付物
- 服务端发电模块与统一接口实现。
- 关键规则测试覆盖（功率计算、燃料消耗）。
- 若涉及服务端 API 行为变更，更新 `docs/dev/服务端API.md`。

## 完成情况
- 新增统一的发电类型与燃料规则模型，支持风能/太阳能/火力/聚变/人工恒星的发电计算与燃料消耗。
- 发电输出写入 `WorldState.PowerInputs`，并在 Tick 中结算发电与能量资源增量。
- 补齐风力/火力/聚变/人工恒星运行参数与燃料输入端口定义，太阳能补充发电类型标记。
- 新增发电功率计算与燃料消耗测试用例。
