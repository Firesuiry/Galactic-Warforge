# T021 生产加成与喷涂系统

## 需求细节
- 加速与增产规则：倍率、持续时间、适用范围与叠加规则。
- 喷涂影响与消耗：喷涂物品类型、消耗率、失败/不足处理。
- 与配方/产出数据的接口约定，确保可被生产链调用。
- 默认参数与可配置化设计。
