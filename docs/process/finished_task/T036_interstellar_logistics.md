# T036 星际物流

## 需求细节
- 星际物流：运输船容量、航速、能耗、翘曲驱动与翘曲器消耗。
- 物流站点：星际设置、出入库规则与缓存。
- 物流调度（星际部分）：航线成本、能耗与供需匹配。

## 交付物
- 服务端星际物流站与运输船运行逻辑实现。
- 单元测试或集成测试覆盖翘曲驱动、能耗与供需匹配。
- 若涉及服务端 API 行为变更，更新 `docs/dev/服务端API.md`。

## 实现结果
- 新增星际物流运输船模型与注册管理，覆盖容量、航速、能耗、翘曲驱动与翘曲器消耗等状态。
- 物流站点补充星际配置与独立供需设置/缓存，库存刷新时同步更新星际缓存。
